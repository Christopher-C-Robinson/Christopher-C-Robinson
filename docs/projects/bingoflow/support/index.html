Support

Send a private support request

Use this form for bug reports, feature requests, or account questions. Submissions go to a private tracker in the private Bingo repo, not a public issue board.

Before you send

Helpful details

  • What you were doing when the issue happened.
  • App version and build number.
  • Device model and iOS version.
  • Whether the issue happened in Scan, Manual Entry, Random, or pattern selection.
  • Any screenshot or screen recording that shows the problem.

Requests stay private. If you include an email address, it is used only for follow-up on that request.
